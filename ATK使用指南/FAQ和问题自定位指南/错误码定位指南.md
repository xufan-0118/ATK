# 507035

**报错原因**：用户在自定义api中修改输入值后，没有将输入tensor搬运到device侧，导致一部分tensor在host侧，一部分tensor在device侧

**解决方法**：
1. `tensor.to(device)`
2. 禁用torch资源池：设置环境变量`export PYTORCH_NO_NPU_MEMORY_CACHING=1`或`os.environ["PYTORCH_NO_NPU_MEMORY_CACHING"] = "1"`

# 507018

**报错原因**: aicpu执行异常，ACL_ERROR_RT_AICPU_EXCEPTION=507018，大概率是算子内部运行超时导致的core dump

**解决方法**
1. 查看plog日志（/root/ascend/log/debug/plog），找对应算子开发或CANN定位解决
2. 检查输入构造是否正确，大概率是kernel执行失败

# 507015

**报错原因**: aicore执行异常，原因和解决方法同507018

# 161001、161002

**报错原因**: 一般为算子的预期报错，配合报错信息定位原因

**解决方法**
1. 如果是该条case下的预期报错，可通过在json用例中设置error_msg参数规避
2. 用户检查自定义api是否引入问题
